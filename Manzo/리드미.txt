* Team : 상린 (Sangrin)

* Game : 만조 (Manzo)

* Version : Spring 2025 Play Test

* 조작법
  + 마우스 좌클릭 : 박자에 맞춰서 누르면 움직임. 연타 가능
  + Tab : Load next game state : 집감, 해수면에서만 갈 수 있으니까 위로 올라가서 누르셈
  + E : 보스 생성
  + R : 보스전 시작 : (꾹 누르기)

집에서 아래에 있는 ship(노랭이)클릭하면 게임 시작할 수 있음
죽으면 화면 까매짐. 클릭하면 집에서 부활

* 게임 시작 방법

- Manzo-Release-Manzo.exe 실행

* 게임 플레이 안내
 맵을 탐험하면서 곳곳에 있는 물고기와 보물상자를 수집하세요.
 바닷속에 살고있는 몬스터를 피하고, 더 아래로 내려가 보스를 발견하세요.
 보스의 공격을 피해 최대한 오래 버티세요. 보스나 몬스터를 공격할순 없습니다.

*보스
신비로운 물고기는 당신을 향해 작은 거품들을 뱉고 원으로 표시된 안전구역을 안내해줍니다.
피하고, 원안으로 들어가 보스를 파훼하세요.

-------------------------------------집에서만 쓸 수 있는 키----------------------------------------
  + O : 물고기 전부 팔기
  + F1 : decide to sell today's fish (-1 -10 +1 +10)

* 몬스터
 일정 거리 가까이 가면 플레이어를 향해 돌진. 잘 피해서 탐험을 이어가세요.

* 보스전
 맵을 탐험하다보면 신호 소리가 들리는 곳을 발견할 것입니다.
 E를 눌러 보스를 소환한 뒤, R을 꾹 눌러 보스전을 시작하세요.
 보스가 발사하는 탄막을 피해서 최대한 오랫동안 생존하세요.
